### Pandas와 Pyspark의 차이

1. Pandas를 사용하는 경우
- 데이터의 크기가 컴퓨터의 RAM 용량보다 작을 때 (보통 수 GB 이내)
- 개인 PC나 단일 서버에서 빠르게 가공하고 분석할 때.
- 대부분의 데이터 사이언티스트가 분석 초기 단계(EDA)에서 가장 많이 사용합니다.

2. PySpark를 쓰는 경우 (Large to Huge Data):
- 데이터 크기가 메모리 용량을 넘어설 때 (수십 GB ~ 수 TB 이상).
- 데이터가 한 대의 컴퓨터가 아니라 여러 서버에 나뉘어 저장되어 있을 때.
- 기업용 데이터 파이프라인이나 실시간 대용량 처리가 필요할 때 필수적으로 사용합니다.

> 주로 실무에서 대규모 데이터를 처리할 때는 PySpark를 사용하기 때문에 반드시 알아둬야하는 라이브러리입니다.
